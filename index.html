BETTA SPLENDENS — FISH CARDIOVASCULAR ANALYSIS
🐟ระบบหมุนเวียนเลือดของปลากัด
ปลากัดมีระบบหมุนเวียนเลือดแบบปิด เลือดไหลภายในหลอดเลือดตลอดเวลา หัวใจมี2 ห้อง คือ Atrium และ Ventricle ต่างจากสัตว์เลี้ยงลูกด้วยนมที่มี 4 ห้อง
🔴หลอดเลือดแดง (Artery)
หน้าที่: นำเลือดออกจากหัวใจไปเลี้ยงอวัยวะ
โครงสร้าง 3 ชั้น:
• Tunica Intima — Endothelium ชั้นในสุด
• Tunica Media — Smooth muscle + Elastin หนามาก
• Tunica Adventitia — เนื้อเยื่อเกี่ยวพันชั้นนอก
แรงดัน: สูงมาก | สี: แดงสด
🟢หลอดเลือดฝอย (Capillary)
ขนาด: 5–10 μm เล็กกว่าเส้นผม ~10 เท่า
ผนัง: 1 ชั้น Endothelium เพื่อให้สารซึมผ่านได้
แลกเปลี่ยน: O₂ + Glucose → เนื้อเยื่อ | CO₂ + Waste → เลือด
ความเร็ว: ช้าที่สุด
🔵หลอดเลือดดำ (Vein)
หน้าที่: นำเลือดกลับหัวใจ
ลักษณะ: ผนังบางกว่า Artery มีValve ป้องกันการไหลย้อน
สี: แดงคล้ำ (Hb + CO₂)
📊ตารางเปรียบเทียบ
คุณสมบัติ Artery Capillary Vein
ทิศทาง ออกจากหัวใจ — เข้าสู่หัวใจ
ผนัง หนามาก 1 ชั้นเซลล์ ปานกลาง
แรงดัน สูง ต่ำ ต่ำมาก
ขนาด เล็ก–กลาง 5–10μm ใหญ่
วาล์ว ไม่มี ไม่มี มี ✓
O₂/CO₂ O₂ สูง กำลังแลก CO₂ สูง
❤️เส้นทางการไหลเวียนในปลา
หัวใจ → Artery → เหงือก → Dorsal Aorta → Capillary → Vein → Atrium → วนซ้ำ

Single Circulation: ผ่านหัวใจ 1 รอบ/วงจร ต่างจาก Double Circulation ของมนุษย์
